Amharic Keyboard (Pro)

A deeply integrated, highly modernized m17n input schema for typing Amharic effortlessly on Ubuntu and Kali Linux architectures.

Traditional Linux Amharic setups suffer from complex configuration processes and bloated phonetic databases that make intuitive typing incredibly frustrating for developers and modern users. Amharic Pro solves this by synthesizing the massive official `m17n` SERA database with a newly engineered, lightweight, logical phonetic mapping.

Under the hood: It utilizes the robust `ibus-m17n` engine, meaning it runs blazingly fast natively within the OS's window manager without relying on third-party daemon processes.

---

Installation Guide

We provide a shell script that handles package management dependencies (`ibus`, `ibus-m17n`, `m17n-db`) and registers the custom `.mim` mapping file to your system tree (`/usr/share/m17n/`).

  1. Ensure executable permissions

    Give the installer execution rights using `chmod`.

    chmod +x install.sh
  2. Execute the script

    Run the script. You will be prompted for `sudo` to use the APT package manager and write to `/usr/share/`. 

    ./install.sh
  3. Restart the display manager

    While the script restarts the `ibus` daemon, it is highly recommended to Log Out and Log In again to ensure GNOME/XFCE input layers refresh.

Configuration

To enable the linguistic mapping in your desktop environment:

  1. Navigate to Settings > Keyboard (or Region & Language depending on your DE).
  2. Under Input Sources, click the + button or "Add Input Source".
  3. Click the ⋮ (More) button and use the search bar to query Amharic.
  4. Select Amharic (Pro) or Amharic (Custom) from the expanded list.
  5. Use the global shortcut (Super + Space) to instantly toggle the input method layer.

2. Advanced Consonant Modifiers

By default, attaching a vowel to a consonant generates the respective Fidel shape (e.g., ka outputs ካ, kee outputs ኬ). To optimize for unique letters, we've formalized intuitive roots:

Root String Outputs Example Output
sz + [vowel] ሠ / ሡ / ሣ sza → ሣ
tz + [vowel] ፀ / ፁ / ፃ tzu → ፁ
zh + [vowel] ዠ / ዡ / ዣ zhe → ዠ
S + [vowel] (Capital) ጸ / ጹ / ጻ Sa → ጻ
T + [vowel] (Capital) ጠ / ጡ / ጣ Ta → ጣ
P + [vowel] (Capital) ጰ / ጱ / ጳ Pa → ጳ
C + [vowel] (Capital) ጨ / ጩ / ጫ Cee → ጬ

3. "Wa" Combinators
